# Weather App

Une application simple pour afficher la météo actuelle d'alger.

## Fonctionnalités

- Affichage de la température, humidité, et conditions météo
- Interface utilisateur intuitive

## Installation

```bash
git clone https://github.com/votre-utilisateur/wether_app.git
cd wether_app
Installez les dépendances selon votre stack (ex: npm install, pip install -r requirements.txt)
```

## Utilisation

```bash
Lancez l'application selon votre stack (ex: npm start, python app.py)
```

## Configuration

- Ajoutez votre clé API météo dans le fichier de configuration approprié.
